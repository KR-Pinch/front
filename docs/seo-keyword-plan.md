- `오늘의 핫토픽`
- `오늘의 토론`
- `1인 1의견`
- `선택된 의견`
- `의견 아카이브`
- `댓글 랭킹`
- `의견 랭킹`

## 페이지별 키워드

- 홈: `오늘의 PINCH`, `오늘의 핫토픽`, `한국 토론 플랫폼`, `1인 1의견`
- 오늘의 PINCH: `오늘의 토론 주제`, `실시간 의견`, `이슈 토론`, `PINCH 토론`
- 아카이브: `PINCH 아카이브`, `토론 아카이브`, `선택된 의견`, `베스트 댓글`
- 랭킹: `PINCH 랭킹`, `댓글 랭킹`, `의견 랭킹`, `주간 랭킹`, `월간 랭킹`

## Search Console에서 봐야 할 순서

1. `site:pinch.kr`로 색인 여부를 먼저 확인합니다.
2. Search Console의 실적 메뉴에서 `오늘의 PINCH`, `pinch`, `pinch.kr` 노출 여부를 봅니다.
3. 노출은 있는데 클릭이 없으면 title/description을 더 선명하게 고칩니다.
4. 노출 자체가 없으면 sitemap 재제출과 URL 검사 색인 요청을 먼저 합니다.
